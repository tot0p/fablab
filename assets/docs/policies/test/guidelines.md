# Test Guidelines

This is a test file to verify that nested navigation works correctly.

## Testing Process

1. Check that parent folders become dropdowns
2. Verify that nested folders also become dropdowns  
3. Ensure all files are accessible through navigation
4. Test that multiple levels of nesting work properly

## Expected Behavior

- The "Policies" folder should be a dropdown
- Within "Policies", the "Test" folder should be another dropdown
- Both "Equipment Reservation" and "Test Guidelines" should be accessible
