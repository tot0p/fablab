# Getting Started with Fablab

Welcome to our Fablab! This guide will help you get started with digital fabrication and using our equipment.

## What is a Fablab?

A Fabrication Laboratory (Fablab) is a digital fabrication workspace where anyone can make almost anything. Our lab is equipped with:

- 3D printers
- Laser cutters
- CNC machines
- Electronics prototyping tools
- Traditional hand tools

## Safety First

Before using any equipment, you must:

1. Complete the safety orientation
2. Read all equipment manuals
3. Get certified on specific machines
4. Always wear appropriate safety gear

## Your First Project

We recommend starting with a simple project like:

- Laser cutting a keychain
- 3D printing a small object
- Basic electronics breadboarding

Ready to dive in? Check out our [Tools & Equipment](/docs/tools/) section for detailed guides on each machine.
