# Electronics Station

Our electronics station provides everything you need for prototyping and building electronic circuits.

## Available Equipment

### Soldering Station
- **Weller WES51**: Temperature-controlled soldering iron
- **Hot air rework station**: For SMD components
- **Desoldering pump**: For component removal
- **Flux and solder**: Various types available

### Measurement Tools
- **Digital multimeters**: Voltage, current, resistance measurement
- **Oscilloscope**: Rigol DS1054Z 4-channel
- **Function generator**: Arbitrary waveform generation
- **Power supplies**: Variable DC voltage/current

### Prototyping Supplies
- **Breadboards**: Various sizes
- **Jumper wires**: Male-male, female-female, male-female
- **Resistors**: 1% metal film, various values
- **Capacitors**: Ceramic, electrolytic, tantalum
- **LEDs**: Various colors and sizes
- **Microcontrollers**: Arduino, Raspberry Pi, ESP32

## Getting Started

1. **Learn basic electronics** theory and safety
2. **Start with Arduino** for beginners
3. **Use breadboards** for prototyping
4. **Test circuits** before soldering
5. **Design PCBs** for permanent projects

## Safety Guidelines

### Soldering Safety
- Use fume extractor when soldering
- Keep iron in stand when not in use
- Wash hands after handling solder
- Wear safety glasses when desoldering

### Electrical Safety
- Check voltage levels before connecting
- Use appropriate probe ratings
- Never work on live high-voltage circuits
- Understand current limitations of components
